Keyboard Drawing Pad Program

For my Penn State Software Engineering (SWENG) 452W class final project, I used C and SDL2 to create a keyboard drawing pad program. This is a simple keyboard-controlled drawing pad application built with SDL2. Users can draw on a 100x100 grid, move the pen with arrow and diagonal keys, adjust brush size, and change colors. Perfect for experimenting with basic graphics programming in C.
